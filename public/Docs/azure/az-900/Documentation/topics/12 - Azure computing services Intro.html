Azure computing services

Compute services is a category of on-demand services used to build and run cloud-based applications, that facilities running your app in the cloud.

Compute services are also known as Infrastructure-as-a-Service (IaaS).

Resources in Azure are available on-demand in minutes or even seconds (depend on the compute option).

- computing service that we are going to discuss are :-

  • Virtual Machine
  • Virtual Machine Scale Sets
  • Container Instance (ACI)
  • Kubernetes services (AKS)
  • App Services
  • Functions

Azure compute services are the hosting services responsible for hosting and running the application workloads. These include Azure Virtual Machines (VMs), Azure Container Service, Azure App Services, Azure Batch and Azure ServiceFabric.
