Create Resource group and resources

We will use azure portal to create resource group and resources. First we will create a resource group so we can populate resources into it.

1. Creating a Resource Group - Using Portal

You need a resource group to work with resources in azure, so lets go and create a resource group.

1. Go to the menu on the left and select resource groups or type "resource groups" on the search resources bar.

2. The resource groups window will open, click on the +Add button to view "Create resource group" form.

3. The first property is subscription, select your subscription for your billing unit for your entire azure account.

4. Second property is the resource group name, provide a name for your resource group, we are going to type "az-900" as the name of our resource group.

5. third property is the region, here we select where the meta data about the resource group is stored.

6. Finally, we click create to start creating our resource group.

7. As you see below our resource group have been created successfully.

2. Creating a Resource Group - Using CLI

Creating resource group thur command line is easy, we call it Azure CLI. To start working with azure CLI you need to login first using the az login command which will redirect you to Microsoft azure login page. 

  az login

 You have logged in. Now let us find all the subscription to which you have access...

- To create a resource group we use the following command 

  az group create --name az-900-RG --location "West US"
