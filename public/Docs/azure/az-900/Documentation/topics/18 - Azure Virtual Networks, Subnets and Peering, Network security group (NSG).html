- Azure Virtual Networking (vNETs)

Virtual networking enables communication between multiple computers, virtual machines (VMs), virtual servers, or other devices across different office and data center locations.

Virtual networking enables you to group and isolate related systems, networking components are logically isolated and segmented into one or more subnets. vNets enables communication of resources with each other, internet and on-premises.
