A.1 - What is Cloud Computing ?

1. Define Cloud computing

Cloud computing is a model for delivering computing services over the internet. Simply it's like using a network of remote servers hosted on the internet to store, manage and process data rather than local server or personal computer.

Cloud computing lets you choose the power and features you need to run your software. All Servers/PCs are in a cloud provider's data-center instead of physically with you. Cloud only lets you pay fot only the services you use

- Computing services which include common IT infrastructure resources and software such as: -

  • Servers with Virtual Machines and containers
  • Storage (HD, SSD)
  • Databases
  • Networking
  • Software/App
  • Analytics and intelligence
  • Also expand the traditional IT offerings to include things like Internet of Things (IoT), machine learning (ML), and artificial intelligence (AI).

Cloud computing really represents renting resources like CPU, RAM and storage from someone else data-center like cloud provider such as Azure. You only pay for what you use Pay-as-you-go pricing model.

The cloud provider takes care of maintaining the underlying infrastructure for you. Someone else gets to manage the infrastructure for you like maintenance, backup and make sure your OS is up-to-date and also make sure that everything is up and running 24hrs a day.

Cloud computing offers flexible resources and economies of scale, this means if you need to increase your IT infrastructure resources rapidly, you don't have to wait to build a new data-center.

Cloud is not constrained by physical infrastructures the same way that traditional data center is. You can treat cloud resources like you would your resources in your own data center. When you're done using them, you give them back.

2. Common cloud/computing services

A cloud provider offers a hundred of services that are grouped with various types of services. The four most common types of cloud services for IaaS - infrastructure as a Service would be Compute, Networking, Storage and Databases.

- Cloud computing common services :-

  • Compute power: is how much processing your computer can do. compute power is a server consist of CPU, RAM and storage. we use virtual machines or containers that run different OS like windows, linux/unix or web container, and with them we can create apps for web, AI, machine learning and more.
  • Storage: is a volume of data you can store, with unlimited data storage that you need/request overtime. We store structured or unstructured data like files and databases. Cloud delivers a thousand of popular DB technologies. Also azure provides tools that allow you to migrate your data/app to the cloud
  • Networking: After gathering your components, we need to connect them to each other securely which our cloud provide a secure networking functionality. You also can connect azure to your company network thru private connection VPN or ExpressRoute.
  • Analytics: cloud provide a Service for analytics capabilities to review your performance with virtualization and telemetry data of your services.

We know that Cloud computing is really bunch of servers that our application and IT services runs on. Servers are comprised of CPU Processor, RAM Memory and storage HDD or SSD. An Example of Applications and IT services like:

  • Email Servers
  • Web Server (web application)
  • Database
  • FTP Servers

3. What is Microsoft Azure/Azure Services?

Microsoft Azure is a cloud computing platform or cloud provider service that delivers a set of services to help you build solutions to meet your business goals.

  • Azure has a simple web service for hosting and managing your business software solution in virtualized computers fully running and presence in the cloud.
  • Azure provides a cloud-based services like remote storage, CPU compute power, database hosting and centralized account management .
  • Azure offers new capabilities like AI (Artificial Intelligence) and IoT (Internet of Things) services.

4. Cloud Benefits

With cloud you don't need to make up large investment in hardware and spend time provisioning and configure the hardware and software. Also provision exactly the right type and size of computing resources you need to power and run your services.

With cloud you don't need to do that, you can have 10 or 100 servers in minutes/instantly and pay for what your use.

- Cloud provider provide on-demand access to :-

  • Access to unlimited resources and services like compute power, storage and networking components. Also cloud offers things like Internet of Things (IoT), machine learning (ML), and artificial intelligence (AI).
  • Cloud provides speech recognition and other cognitive services that help make your applications stand-out from the crowd
  • Cloud provides analytics services that enable you to track your software with telemetry data coming back from your devices.
  • Cloud provides tools for migrating your apps and data to their virtual machines.
